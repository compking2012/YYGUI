value(visualFormat, options)

Parses a single line of vfl into an array of constraint definitions.

When the visual-format could not be succesfully parsed an exception is thrown containing additional info about the parse error and column position.

Parameters:
Name Type Argument Description
visualFormat {String} Visual format string (cannot contain line-endings!).
options {Object} optional  Configuration options.
Properties
Name Type Argument Description
extended {Boolean} optional  When set to true uses the extended syntax (default: false).
outFormat {String} optional  Output format (`constraints` or `raw`) (default: `constraints`).
lineIndex {Number} optional  Line-index used when auto generating equal-spacing constraints.
Returns:
{Array} Array of constraint definitions.
